我们发布了一个新的 Claude Code 插件，叫做 playground，帮助 Claude 生成 HTML 游乐场。这些是独立的 HTML 文件，让你可以用 Claude 可视化一个问题、与它互动，并给你一个输出提示可以粘贴回 Claude Code。

我发现这对于以不适合文本的方式与模型互动非常有用，例如：

- 可视化代码库的架构
- 调整组件的设计
- 头脑风暴布局和设计
- 调整游戏的平衡

要开始，在 Claude Code 中运行以下命令来安装插件：

/plugin marketplace update claude-plugins-official
/plugin install playground@claude-plugins-official

以下是我最喜欢的一些游乐场：

## 更改 Claude Code 中 AskUserQuestion 工具的设计

提示：「使用 playground skill 创建一个游乐场，帮助我探索 AskUserQuestion 工具的新布局变化」

## 批评你的写作并获得回复

提示：「使用 playground skill 审查我的 [SKILL.MD](https://skill.md/)，并给我我可以批准、拒绝或评论的内联建议」

## 调整 Remotion 视频开场

提示：「使用 playground skill 调整我的开场画面，让它更有趣和更令人愉悦」

## 查看架构图并让用户评论

提示：「使用 playground skill 展示这个邮件 agent 代码库的工作原理，并让我评论架构中的特定节点来提问、提出编辑等」

## 平衡我正在制作的超级英雄类 Rogue 游戏

提示：「使用 playground skill 帮助我平衡'Inferno'英雄的牌组」

很期待看到你们所有人如何探索这个！

我创建有趣游乐场的建议——想一个与模型互动的独特方式，然后让 Claude 表达出来。我觉得你可能会感到惊讶。
