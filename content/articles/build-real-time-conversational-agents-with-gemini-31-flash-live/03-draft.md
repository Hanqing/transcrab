# 使用 Gemini 3.1 Flash Live 构建实时对话 Agent

今天，我们通过 Google AI Studio 的 Gemini Live API 发布 Gemini 3.1 Flash Live。Gemini 3.1 Flash Live 帮助开发者构建实时语音和视觉 Agent，不仅能处理周围世界，还能以对话的速度响应。

这是延迟、可靠性和更自然对话质量的里程碑式提升，提供下一代语音优先 AI 所需的品质。

## 体验增强的延迟、可靠性和质量

对于实时交互，每一毫秒的延迟都会削弱用户期望的自然对话流程。新模型更好地理解语调、重音和意图，实现以下关键改进：

- 在嘈杂的真实环境中更高的任务完成率：我们显著提升了模型在实时对话中触发外部工具和提供信息的能力。通过更好地辨别交通或电视等环境声音中的相关语音，模型能更有效地过滤背景噪音，保持对指令的可靠响应。
- 更好的指令遵循：对复杂系统指令的遵循显著提升。即使对话出现意外转折，你的 Agent 也会保持在运营护栏内。
- 更自然、低延迟的对话：最新模型在延迟方面有所改进，与 2.5 Flash 原生音频相比，在识别音调和节奏等声学细微差别方面更加有效，使实时对话感觉更加流畅自然。
- 多语言能力：该模型支持超过 90 种语言的实时多模态对话。
