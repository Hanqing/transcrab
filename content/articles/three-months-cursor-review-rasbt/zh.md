Three months of Cursor — honest review

I've been using Cursor at my new role for 3 months, and here's my honest take.

**What I actually use most:**
1. ⌘K for inline edits (I still tab-complete with Copilot, but Cursor's codebase-aware suggestions are noticeably better)
2. Cmd+L for chat (love having the full codebase context without constantly alt-tabbing)
3. Cmd+K → /doctor (love this for debugging)

**What I DON'T use:**
- Composer (I find it overwhelming)
- Tab (I keep it off, prefer manual control)
- AI actions (I don't need them)

**What I miss from GitHub Copilot:**
- The COMMAND palette integration (Cmd+Shift+P is so natural for shortcuts)

**Would I pay for it?**
Yes. $20/mo is a no-brainer for me.

The codebase-aware suggestions alone make it worth it — I get fewer irrelevant suggestions and much more contextually relevant completions.
